## 기술 스택

- **프레임워크**: Next.js 14 (App Router)
- **언어**: TypeScript
- **스타일링**: Tailwind CSS + ShadCn UI
- **상태 관리**: Jotai + React Query
- **데이터베이스**: Supabase
- **인증**: Supabase Auth
- **패키지 매니저**: pnpm
- **모노레포 도구**: Turborepo
